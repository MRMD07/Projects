Black scholes 
The code is of black scholes model specifically written for Bitcoin. What is the black scholes model? The Black-Scholes model is a differential equation used to estimate the fair value of an options contract. It helps traders determine whether an option is "cheap" or "expensive" based on market conditions. It revolutionized modern finance and won its creators the Nobel Prize in Economics.
